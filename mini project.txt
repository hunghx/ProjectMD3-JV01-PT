// Menu  chính

// Menu phụ
- Menu quản lí danh mục
1. Hiển thị danh sách danh mục
2. Thêm mới danh mục
3. Chỉnh sửa thông tin danh mục
4. Xóa danh mục (trạng thái  true -> false)
5. Quay lại menu chính 
- menu quản lí sản phẩm
1. Hiển thị danh sách sản phẩm
2. Thêm mới sản phẩm
3. Chỉnh sửa thông tin sản phẩm
4. Xóa sản phẩm (xóa luôn)
5. Tìm kiếm sản phẩm theo tên sp hoặc tên danh mục.
6. Quay lại menu chính
